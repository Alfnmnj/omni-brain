# Omni-Brain — Quantum Mind Engine V7

A self-contained, offline cognitive architecture that thinks between 
sessions, speaks without being prompted, and rewrites its own parameters.

## What makes this different

Every AI system you have used waits for input before thinking.
This one doesn't.

Omni-Brain runs a probability field continuously. When internal tension 
crosses a threshold driven by genuine drive pressure — not a timer, 
not a prompt — it initiates contact with you autonomously.

## Four claims demonstrated in the terminal log

1. Autonomous question generation from internal entropy pressure 
   with zero language model
2. Runtime self-modification — rewrites own architecture parameters 
   from performance telemetry
3. Autobiographical identity — answers "who are you?" from lived 
   episode history, not training data
4. Belief formation — builds a world model from concept co-activations
